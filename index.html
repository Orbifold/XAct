XAct

Swift numerics, maths, data structures, algorithms and machine learning.

View on GitHub Download .zip Download .tar.gz

A comprehensive Swift framework library containing numerical algorithms, mathematical functions, data structures, machine learning algorithms, data generators and more.

The library is ported from the Numerics library for .Net and is part of Orbifold's consultancy toolbox.
